Tic Tac Toe 🎮

A simple Tic Tac Toe game built using Java Swing and AWT. The game allows two players to enter their names and play through a graphical user interface.

Features
Player name setup screen
Interactive 3×3 game board
Alternate turns for X and O
Winner detection
Draw detection
Restart game option
Clean and user-friendly UI
Technologies Used
Java
Swing
AWT
Object-Oriented Programming (OOP)
How to Run
Clone the repository.
Open the project in your Java IDE.
Run PlayerSetup.java.
Enter player names and start playing.
